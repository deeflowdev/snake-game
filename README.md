# 🐍 Snake Game

A classic Snake game built using HTML, CSS, and vanilla JavaScript with smooth gameplay and responsive controls.

## 🎮 Features
- Grid-based snake movement
- Collision detection (walls and self)
- Score tracking system
- Game over state with restart option
- Responsive layout for desktop and mobile
- Smooth keyboard and touch controls

## 🎯 Controls
Use arrow keys (or on-screen controls on mobile):
- ↑ Up
- ↓ Down
- ← Left
- → Right

## 🧠 How it works
The snake grows when it eats food. The game ends when it hits the wall or itself.
